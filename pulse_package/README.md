# Pulse+ Filpakke

Dette arkiv indeholder alle relevante filer til Pulse+ projektet:

- **pulse_module_spec.md** – Opgavebeskrivelsen og systemets formål.
- **project_backlog_detailed.md** – Detaljeret backlog med epics og tasks.
- **pulse_project_code_skeleton.md** – Kodestubbe for backend, frontend og pipeline, inkl. testsektion.
- **tests/** – Prøve testcases:
  - `test_specific_backend.py` – Ét eksempel per område (commit parsing, context, insight, permissions, analytics).
  - `PulseCard.test.jsx` – En React‑test for PulseCard‑komponenten.

Brug disse filer som udgangspunkt for udvikling, test og videre arbejde.
